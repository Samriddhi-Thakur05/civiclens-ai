# CivicLens AI

CivicLens AI is a hackathon-ready solution for **Problem Statement 2: Community Hero - Hyperlocal Problem Solver**.

It helps citizens report local infrastructure issues and turns messy reports into structured, trackable civic cases using Gemini multimodal analysis.

## Core workflow

1. Citizen uploads an issue photo and adds location context.
2. Gemini analyzes the image and notes.
3. The app generates a structured civic case with category, severity, responsible department, duplicate risk, reasoning, and next actions.
4. The community validates the report through confirmations.
5. Officials and citizens can track status and impact through a priority queue and dashboard.

## Google technologies used

- **Google AI Studio Build Mode** for building and deploying the application.
- **Gemini API** for multimodal issue analysis.
- **Structured JSON output** for reliable AI triage.
- **Cloud Run deployment URL** generated by Google AI Studio publishing.

The official AI Studio deployment guide says Build Mode apps can be published directly from AI Studio and receive a Cloud Run URL after deployment.

## Agentic depth

CivicLens AI is designed as a triage agent, not a passive report form. It:

- Classifies civic problems from text and images.
- Estimates urgency and public-safety severity.
- Routes reports to the likely responsible department.
- Compares reports against nearby existing issues for duplicate risk.
- Generates practical next actions for officials and community volunteers.
- Updates public status as the case moves from reported to resolved.

## Run locally

Open `index.html` in a browser, or serve the folder with any static web server.

For live Gemini analysis, paste a Gemini API key into the app. Without a key, the app runs in demo mode so judges can still evaluate the full product flow.

## Deploy with Google AI Studio

1. Open Google AI Studio.
2. Use Build Mode to create/import the app.
3. Add these files to the app: `index.html`, `styles.css`, and `app.js`.
4. Test the report flow with a Gemini API key.
5. Click **Publish**, then **Get Started**, then **Publish App**.
6. Submit the generated Cloud Run URL on BlockseBlock.
